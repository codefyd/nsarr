# نظام مجمع النصار لإدارة الحلق

نظام متكامل لإدارة مجمع حلقات تحفيظ القرآن الكريم، يشمل بوابات للتسجيل، بيانات الطلاب، وبوابة للعاملين مع نظام إنذارات متطور.

## المميزات الرئيسية

### 1. بوابة التسجيل والقبول
- تسجيل الطلاب الجدد بسهولة
- إدخال بيانات الطالب وولي الأمر
- التحقق من صحة البيانات
- إدارة طلبات التسجيل والقبول والرفض

### 2. بوابة بيانات الطالب
- البحث عن بيانات الطالب برقم الهوية أو جوال ولي الأمر
- عرض كل بيانات الطالب
- طلب تعديل البيانات

### 3. بوابة العاملين
- **المعلم:** عرض طلاب الحلقة وإضافة ملاحظات
- **المشرف التعليمي:** إصدار إنذارات تعليمية
- **المشرف الإداري:** إدارة كاملة للنظام
  - إدارة الطلاب (إضافة، تعديل، حذف)
  - معالجة الطلبات الواردة
  - إدارة الإنذارات التعليمية والإدارية
  - الإعدادات والقوائم

### 4. نظام الإنذارات
- **إنذارات تعليمية:** ضعف حفظ، غياب، سلوك
- **إنذارات إدارية:** تأخر، غياب، غياب بعذر
- نظام عتبات قابل للتخصيص
- متابعة إجراءات الإنذارات

## المتطلبات التقنية

- **Google Sheets:** لتخزين البيانات
- **Google Apps Script:** للمعالجة الخلفية
- **متصفح حديث:** يدعم HTML5 و JavaScript

## التثبيت والإعداد

### 1. إعداد Google Sheets

1. أنشئ جدول بيانات جديد على Google Sheets
2. انسخ رابط الملف (سيكون مثل: `https://docs.google.com/spreadsheets/d/SHEET_ID/edit`)

### 2. إعداد Google Apps Script

1. افتح الملف على Google Sheets
2. انقر على **الأدوات** > **محرر البرامج النصية**
3. انسخ محتوى ملف `Code.gs` إلى المحرر
4. احفظ المشروع

### 3. نشر التطبيق

1. انقر على **نشر** > **نشر كتطبيق ويب**
2. اختر:
   - **تنفيذ باسم:** حسابك
   - **الوصول إليه باسم:** أي شخص
3. انسخ الرابط المنشور

### 4. إعداد الملفات الأمامية

1. انسخ رابط التطبيق المنشور
2. افتح ملف `config.js`
3. استبدل `ضع-هنا-رابط-Google-Apps-Script` برابط التطبيق

### 5. نشر الملفات

1. ارفع الملفات التالية إلى موقع ويب:
   - `index.html`
   - `app.js`
   - `config.js`

أو استخدم Google Sites لاستضافة الملفات.
